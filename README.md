# 🎬 Movie & TV Analytics Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze movies and TV shows using ratings, votes, genres, and content type information.

The goal of the dashboard is to provide a clear overview of:
- content performance,
- audience ratings,
- genre popularity,
- distribution of ratings,
- comparison between Movies and TV Shows.

The dashboard was built to improve my Power BI skills while applying data analysis concepts in a real-world style project.

---

# 📊 Dashboard Features

## KPI Cards
The dashboard includes key performance indicators for both Movies and TV Shows:
- Total Video Count
- Average Rating
- Total Votes

These KPIs allow quick comparison between content types.

---

## Genre Analysis
A bar chart shows the average rating per genre, helping identify:
- highest rated genres,
- lowest rated genres,
- audience preferences.

---

## Rating Distribution
A histogram-style visualization displays how videos are distributed across rating buckets.

This helps understand:
- concentration of ratings,
- overall quality distribution,
- presence of extreme ratings.

---

## Top Rated Content
A ranked table displays:
- movie/show posters,
- average rating,
- number of votes.

This section highlights the most appreciated titles by users.

---

## Interactive Filters
Users can filter the dashboard by:
- Movie
- TV Show

This allows dynamic exploration of the dataset.
